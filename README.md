# movie-theater-api
Movie Theater Project for Express Week Day 5

## Getting Started

```bash
$ npm install
```

## Start the server
To start the server

```bash
$ npm run start
```

## Host the web app
To host the web app
```bash
$ npm run web
```

## Running Tests
To run tests, stop both web app and server.

Comment the following lines in server.js:
```javascript
// app.listen(5001, () => {
//   seed();
//   console.log("Listening on port 5001");
// });
```

```bash
$ npm test
```
